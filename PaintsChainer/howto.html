How to Use

1

Select a pending color line image or drawing image

2

If you use the line drawing service, please click the "AI Colorization" button after uploading the picture. atter selecting an image

3

If you use color guide customization to generate a color map, click the "AI Colorization" button after adding hint information using color pen in the left image.

4

If you want to use the color picture transfer service, please upload the color picture and click the "AI sketch" button.
